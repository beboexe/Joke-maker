# Joke-maker
A beginner‑friendly Joke Maker web app built with HTML, CSS, and JavaScript. It demonstrates the use of callbacks, promises, async/await, and the Fetch API by fetching random jokes from a public API and displaying them on the page.

# Joke Maker App 🎭

A simple beginner‑friendly web app that fetches random jokes using the **Fetch API**.  
This project is designed to help beginners understand **callbacks**, **promises**, and **async/await** in JavaScript.

---

## 🚀 Features
- Click a button to fetch a random joke.
- Shows a loading message while fetching.
- Displays the joke neatly below the button.
- Demonstrates both `.then()` and `async/await` approaches.

---

## 🛠️ Technologies Used
- **HTML** → Structure
- **CSS** → Styling
- **JavaScript** → Logic (callbacks, promises, async/await, fetch)
